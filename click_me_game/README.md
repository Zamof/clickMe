# Click Me Game 🎮

A fun and engaging Flutter mobile game where you need to click the right buttons while avoiding the wrong ones!

## 🚀 New Features & Improvements

### ✨ Enhanced Gameplay
- **Lives System**: Start with 3 lives, lose one for each wrong click
- **Level Progression**: Difficulty increases as you score more points
- **Power-ups**: Random power-up activation with visual indicators
- **Bonus Lives**: Earn extra lives every 10 points scored

### 🎨 Visual Enhancements
- **Smooth Animations**: Button scale, score pop, and shake effects
- **Particle Effects**: Beautiful particle explosions for clicks and misses
- **Modern UI**: Gradient backgrounds, rounded corners, and shadows
- **Responsive Design**: Adapts to different screen sizes
- **Better Colors**: More vibrant and visually appealing color scheme

### 🎯 Game Mechanics
- **Dynamic Difficulty**: Button size decreases and speed increases with level
- **Smart Positioning**: Buttons appear within screen bounds
- **Pause Functionality**: Pause and resume the game anytime
- **Enhanced Feedback**: Visual and audio cues for all actions

### 🔧 Technical Improvements
- **Performance Optimized**: Efficient animation controllers and state management
- **Better Code Structure**: Cleaner, more maintainable code
- **Material 3 Design**: Modern Flutter design principles
- **Responsive Layout**: Works on all device orientations

## 🎮 How to Play

1. **Click "Click me"** buttons to score points
2. **Avoid "Don't click me"** buttons to keep your lives
3. **Score 10 points** to level up and earn bonus lives
4. **Use pause button** to take breaks
5. **Beat your high score** and challenge friends!

## 🛠️ Installation

1. Ensure you have Flutter installed on your system
2. Clone this repository
3. Navigate to the project directory
4. Run `flutter pub get` to install dependencies
5. Connect a device or start an emulator
6. Run `flutter run` to start the game

## 📱 Requirements

- Flutter SDK 3.8.1 or higher
- Dart SDK 3.0.0 or higher
- Android 5.0+ or iOS 11.0+
